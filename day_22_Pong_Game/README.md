# 🏓 Pong Game

## 📖 Description

A classic Pong game built using Python's Turtle module. This project
demonstrates core programming concepts such as object-oriented
programming, game loops, collision detection, and real-time user input
handling.

------------------------------------------------------------------------

## 🚀 Features

-   Two-player gameplay
-   Smooth ball movement with wall bouncing
-   Paddle collision detection (full paddle coverage)
-   Center dashed line UI
-   Real-time score tracking
-   Keyboard controls for both players

------------------------------------------------------------------------

## 🛠️ Tech Stack

-   Python 3
-   Turtle (standard library)

------------------------------------------------------------------------

## 🎮 Controls

  Key   Action
  ----- -------------------
| Key | Action |
|-----|--------|
| W | Left paddle up |
| S | Left paddle down |
| ↑ | Right paddle up |
| ↓ | Right paddle down |
