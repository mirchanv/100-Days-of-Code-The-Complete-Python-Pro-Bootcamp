# 🔤 NATO Phonetic Alphabet Converter

A Python program that converts any user-input word into its corresponding NATO phonetic alphabet representation.

---

## 🚀 Features

- 📄 Reads phonetic data from a CSV file  
- 🔄 Converts letters into NATO phonetic code words  
- 🧠 Uses dictionary comprehension for efficient mapping  
- 🔤 Handles case-insensitive input  
- ⚡ Simple and fast execution  

---

## 🛠 Tech Stack

- Python 3  
- Pandas (for CSV data handling)
